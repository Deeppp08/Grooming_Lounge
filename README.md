# Apex Grooming Lounge

A premium men's grooming and salon website designed to deliver a modern, luxurious, and seamless digital experience. The platform showcases professional grooming services, pricing plans, customer testimonials, and appointment booking features through an elegant and responsive user interface.

1. Clone the repository:

```bash

```

2. Open the project folder:

```bash

```

3. Run the project by opening `index.html` in your browser.
